# Swift Family

A growing collection of modular, production-ready Swift packages for Apple platform developers — built from real problems, free forever.

## Philosophy

Every package in this organization started as something we needed ourselves. Instead of keeping solutions buried in private codebases, we extract, clean, and open-source them as standalone Swift packages so anyone can drop them in and move on.

- **Modular** — each kit does one thing well
- **Production-ready** — used in real apps before being published
- **MIT licensed** — use freely in personal and commercial projects
- **No dependencies** — pure Swift, no bloat

## Packages

## Usage

All packages are available via Swift Package Manager. Add the package URL for any kit through Xcode → File → Add Package Dependencies.

```
https://github.com/swift-family/<package-name>
```

## Contributing

Found a bug? Have a feature idea? PRs and issues are welcome on any repository.

If you have a Swift utility you'd like to contribute as a new kit, open a discussion — we're happy to review and help shape it into a proper package.
